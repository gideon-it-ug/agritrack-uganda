# 🌾 AgriTrack Uganda

> Live crop price tracker across Uganda's districts

## What it does
AgriTrack helps farmers and traders check real-time crop prices
across Uganda — so no one gets exploited by middlemen.

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** Django + Django REST Framework
- **Database:** PostgreSQL
- **SMS Alerts:** Africa's Talking API

## Features (in progress)
- [x] Price cards by crop and district
- [ ] Django REST API
- [ ] PostgreSQL database
- [ ] Uganda district map
- [ ] SMS price alerts
- [ ] AI price predictions

## Running locally
\`\`\`bash
# Frontend
cd frontend
npm install
npm run dev
